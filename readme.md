## Website Boilerplate

In this bundle are included :
- TailwindCSS
- JQuery
- CleaveJS (Input formating)
- AlpineJS
- Wow JS
- Confetti JS
- Tailwind Forms
- Tailwind Typography
- Tracy (PHP Debuging)
- Respect/Validation (Forms)
- Parsedown (Extra included)
- PHP Mailer
